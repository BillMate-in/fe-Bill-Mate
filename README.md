# BillMate - Shared Dining Experience App

BillMate adalah sebuah aplikasi web modern dan interaktif yang dirancang khusus untuk mempermudah pengelolaan pesanan, pembagian tagihan (*split bill*), dan pencatatan riwayat transaksi saat makan bersama. Aplikasi ini menjadi solusi cerdas untuk menghitung totalan biaya tanpa drama, cepat, akurat, transparan, serta bebas dari kerumitan kalkulator manual.

## 🚀 Fitur Utama

- **Membuat Ruang Pesanan (Create Room):** Memulai sesi baru dengan memasukkan nama host, nama restoran, dan nomor rekening tujuan pembayaran untuk mengunci koordinasi grup.
- **Manajemen Pesanan Interaktif (Live Room Ordering):** Mencatat dan menambahkan menu makanan atau minuman, menentukan harga, serta menetapkan siapa anggota grup yang memesan menu tersebut secara interaktif.
- **Kalkulasi Biaya Otomatis (Live Billing System):** Menghitung subtotal secara langsung yang terintegrasi dengan penyesuaian komponen tambahan seperti persentase pajak (PPN), diskon toko, hingga biaya ekstra (ongkir/parkir).
- **Nota Digital Ekspor (Digital Receipt):** Menghasilkan rincian nota final yang membagi proporsi tagihan secara adil berdasarkan pesanan masing-masing individu, lengkap dengan nomor rekening host yang dapat disalin serta tombol integrasi kirim rincian langsung ke WhatsApp.
- **Pencatatan Riwayat (History & Search):** Menyimpan setiap sesi makan bersama secara otomatis ke dalam daftar riwayat yang dilengkapi dengan filter pencarian nama sesi, penyaringan waktu (7 hari dan 30 hari terakhir), serta fitur navigasi halaman (*pagination*).
- **Desain Adaptif Responsif:** Antarmuka visual yang bersih, modern, dan nyaman diakses secara optimal baik melalui perangkat HP maupun komputer desktop.

## ⚙️ Cara Menjalankan Aplikasi

Aplikasi BillMate berjalan sepenuhnya di sisi klien (*client-side frontend*), sehingga Anda dapat langsung mencobanya tanpa perlu proses instalasi server, database, atau dependensi backend yang rumit:

1. **Persiapan Berkas:** Pastikan seluruh file dokumen aplikasi berada di dalam satu direktori/folder kerja yang sama.
2. **Akses Lewat Browser:** Klik dua kali pada file `index.html` untuk membuka dan menjalankan aplikasi pada browser pilihan Anda (seperti Google Chrome, Microsoft Edge, Mozilla Firefox, atau Safari).
3. **Rekomendasi Pengembangan:** Jika Anda menggunakan *Visual Studio Code*, jalankan melalui ekstensi **Live Server** agar halaman otomatis diperbarui secara *real-time* setiap kali ada perubahan kode yang disimpan.
